# ModMultiVersionTool

命令行版的[ModMultiVersion](https://github.com/kitUIN/ModMultiVersion)

- 不支持反向更新


## 使用方法

在项目目录直接启动
- 示例: [init.ps1](https://github.com/kitUIN/ChatImage/blob/master/init.ps1) && [tool](https://github.com/kitUIN/ChatImage/tree/master/tool)
